Detailed Description

Collection of abstract classes forming the basis of clean eco-system on .NET Core 3.0

Enterprise Members of products' computing ecosystem are developed from this clean, reusable, cross-platform core. Enterprise Members may include:

  • An Engineering GUI – for developer's use
  • A Technicians' Assistant – for use by service staff
  • Production Line GUI(s) – for various use by production crew
  • Quality/Regulatory Database tools
  • Applications for customers and end-users
